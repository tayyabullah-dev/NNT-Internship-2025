What is JavaScript and how is it used?
JavaScript is a programming language used to make web pages interactive. It runs in the browser and allows developers to add features like animations, form validation, and dynamic content updates. It can also be used on the backend with Node.js.
What is the difference between let, const, and var?
var: Function-scoped, can be redeclared, hoisted with undefined.
let: Block-scoped, cannot be redeclared in the same.
const: Block-scoped, cannot be redeclared or reassigned.
What is an arrow function in JavaScript?
An arrow function is a shorter way to write a function using =>. It does not have its own this value.
What is the difference between == and === in JavaScript?
== (Loose Equality): Compares values without checking types.
=== (Strict Equality): Compares both values and types.
What is the fetch API in JavaScript?
The fetch API is used to make HTTP requests in JavaScript.This fetches data from an API and logs it to the console.
